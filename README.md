# Online Shopping Cart

這是一個模擬的線上購物車應用程式，允許使用者瀏覽商品、選擇尺寸和顏色、加入購物車，並完成訂單。

## 功能列表

- 瀏覽商品列表
- 按分類篩選商品
- 查看商品詳情（選擇尺寸和顏色）
- 添加商品到購物車
- 查看和編輯購物車
- 結帳流程（模擬）

## 部署網址

## 技術棧

- 前端框架: React
- UI 框架: Material-UI
- 狀態管理: React Context API
- 路由: React Router
